# PlaywrightAutomation project

> 1. Run test no UI
```Javascript
yarn playwright test name_file.spec.js --project=chromium
```

>2. Run test with UI
```JavaScript
yarn playwright test VerifyLogin.spec.js --project=chromium --headed
```

### Tài liệu
- Tham khảo: [Cài đặt](https://playwright.dev/docs/intro)
